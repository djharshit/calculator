# GUI Calculator
#### Made by DJ Harshit

---

### How to use it ?

Run the program and calculator start.
* You can use mouse button click on the
buttons for the expression

**OR**

* Use numeric keys of keyboard for input.

|   Key   |     Function      |
|  -----  | ----------------- |
|  0...9  |      numbers      |
|  +-*/   |     operators     |
|  Enter  |      equal        |
|Backspace|      back         |
|  Delete |      clear        |
